# 사용자지정 파일탐색기

오픈소스 대회 사용자지정 파일탐색기 프로젝트

## 프로젝트 목표

- 고속처리
- 기존에 없던 유용한 기능 : 멀티윈도우 / 향상된 즐겨찾기 등
- 적절한 추상화를 통한 손쉬운 오픈소스 기여


## 사용기술 / tool

- rust + tauri
- sveltekit
- vscode

## 아키텍쳐

- 추가예정

## Building

프론트엔드 개발서버

```bash
npm run dev
```

프론트엔드 빌드
```bash
npm run build
```

개발 앱 빌드
```bash
cargo tauri dev
```
